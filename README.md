# Discord 자판기 봇

## 📋 프로젝트 설명
디스코드 자판기 봇 및 웹 대시보드 - 포인트 충전, 제품 구매, 자동 역할 지급등 다양한 기능

## 🚀 시작하기

### 1. 패키지 설치
```bash
npm install
```

### 2. 환경 변수 설정

### 3. 데이터베이스 초기화
```bash
npx prisma generate
npx prisma db push
```

### 4. 실행
```bash
npm run dev
```

## 📁 프로젝트 구조

```
src/
├── commands/          # 슬래시 명령어
│   └── slash/        # 슬래시 명령어 파일
├── events/          # 디스코드 이벤트
├── handlers/        # 핸들러
├── utils/           # 유틸리티
├── dashboard/       # 웹 대시보드
│   ├── routes/      # 라우트
│   ├── views/       # EJS 템플릿
│   ├── middleware/  # 미들웨어
│   └── public/      # 정적 파일
└── index.js         # 메인 파일

prisma/
└── schema.prisma   # 데이터베이스 스키마
```

## ⚙️ 주요 기능

### 🤖 봇 명령어
- `/수동충전 [유저] [금액]` - 수동 포인트 충전
- `/잔액보기 [유저]` - 잔액 조회
- `/웹패널` - 대시보드 링크
- `/임베드게시 [채널]` - 상점 임베드 게시

### 💰 충전 시스템
- 입금 버튼 클릭 → 모달 → 입금자명/금액 입력
- SMS 웹훅 연동 시 자동 충전
- 중복 결제 방지 로직

### 🛒 제품 구매
- 고정형: 구매 시 모든 콘텐츠 지급
- 재고형: 구매 시 랜덤 재고 1개 지급
- DM 발송 (실패 시에도 영수증에 저장)

### 🎭 자동 역할 지급
- 누적 구매량 기준 역할 자동 지급
- 웹 대시보드에서 규칙 관리

## 📱 SMS 자동충전 설정

### 안드로이드
1. SMS Forwarder 앱 설치
2. Web Sender 생성 → SMS_WEBHOOK_URL 입력
3. SMS 수신 시 웹훅 전송 규칙 설정

### 아이폰
1. 단축어 앱 → 자동화
2. 메시지 트리거 추가
3. GET/POST로 웹훅 URL 전송
